*** PROJ Ideia de estrutura de QCA <2021-03-20 sáb>

- Casos :: Países
- Variáveis :: Conflitos de acordo com alguma periodização (/e.g./ Guerra da Coreia)
- Resultado :: Tendência da distribuição no período de análise
- Tempo :: Se houver sobreposição dos conflitos, pode ser o caso de dividir em mais de um QCA ou agregar as variáveis
- Fuzzy :: A variável pode ser calibrada de acordo com a distância da respectiva capital
- Direcionais :: Todos contribuiriam para o resultado
- Análise :: Foco na suficiência e necesidade; menos na equação minimizada
*** PROJ Ideia de estrutura de QCA <2021-03-29 seg>

As variáveis explicativas são as mesmas do heading anterior, a diferença é o design do modelo.
A ideia é fazer mais de um QCA em subperíodos diferentes e selecionar os conflitos relevantes a partir do processo de minimização.
Estes QCA não têm um fim por si só, o propósito é apenas reduzir o número de dimensões.
Os subperíodos devem ser selecionados por critérios históricos.
Caso não exista um critério bem definido, pode-se utilizar testes de quebra estrutural como proxy.
Com os casos relevantes selecionados, analiza-se a necessidade e suficiência deles para com o resultado.
O processo de minimização nesta etapa é menos relevante.

Para conseguir estruturar este QCA, propõe-se os seguintes dataframes:
- Descrição dos conflitos :: Inclui as informações relevantes de cada conflito como ano de início e fim, duração, países envolvidos e subperíodo ao qual pertence.
  + Sugere-se que cada conflito possua um código e uma descrição mais detalhada
  + A duração serviria para propóstos de programação apenas
  + O subperíodo também pode ser utilizado para filtrar os meta-QCAs
  + Esta tabela pode dar uma sugestão de narrativa dos capítulos anteriores.
- Países e os conflitos :: Os países são os títulos das linhas, os conflitos são as colunas e as entradas apresentam a distância deste conflito da capital
- Variáveis macroeconômicas :: Países e anos com variáveis macroeconômicas. Esta tabela deve incluir o resultado e, se for útil, outras variáveis macroeconômicas de interesse.
- Meta-QCAs :: É a junção das informações anteriores a partir de filtros estabelecidos. Pode-se elaborar uma função para gerar tais tabelas. O objetvo final é gerar vetores com os nomes de conflitos relavantes (para ausência e presença do resultado) para então estimar o modelo completo.
