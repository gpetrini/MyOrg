* Introdução à teoria das probabilidades


** Introdução

- Formulação matemática de probabilidade depende de teoria dos conjuntos
  - União (\(\cup\)): \(A\cup B=\{x\,\mid\, x \in A \text{ e/ou } x \in B\}\)
    - Também inclui a ocorrência de ambos
  - Intersecção (\(\cap\)): \(A\cap B=\{x\,\mid\, x \in A \text{ e } x \in B\}\)
  - Complementar (\(A^{c}\) ou \(B \setminus A\)): \(A^{c} = \{x \in B | x \notin A\}\)
  - Exemplos:
    - Evento A ocorre, mas B não ocorre: \(A \cap \overline{B}\)
    - Nenhum evento ocorre: \((A \cup B)^{c}\)
- Experimento aleatório: Experimento que pode ser repetido inúmeras vezes sob as *mesmas condições*, sendo o resultado incerto
- Definições:
  - Espaço amostral (\(\Omega\)): Representa o conjunto de todas as *saídas* possíveis de um experimento aleatório
    - Exemplo: dado (\(1 \ldots 6\))
  - Evento (\(A\)): Subconjunto do espaço amostral
  - Evento impossível (\(\emptyset\)): evento que nunca ocorre (conjunto vazio)
    - Exemplo (dado): \(\{x > 6 | x \in \Omega\}\)
  - Evento mutuamente excludente: \(A \cap B = \emptyset\)

** Axiomas de probabilidade

Dadas as definições anteriores, a função \(P(\cdot)\) é denominada probabilidade se satisfazer:
- \(0 \leq P(\cdot) \leq 1, \quad \forall A \in \Omega \)
- \(P(\Omega) = 1\)
- Se \(A_{1}, A_{2}, \ldots \) forem mutualmente excludentes,
  - \(P(\Cup_{i=1}^{\infty} A_i) = P(A_1 \cup A_2 \cup \ldots) = \sum^{\infty}P(A_i)\)
- Probabilidade clássica :: Se um experimento aleatório tiver \(n(\Omega)\) resultados mutuamente exclusivos e igualmente possı́veis, e se um evento \(A\) conter \(n(A)\) desses resultados, a probabilidade de ocorrência desse evento é definida por:
  - \(P(A) = n(A)/n(\Omega) = |A| / |\Omega|\)
  - Só é válida se os eventos forem equiprováveis
- Interpretação frequentista :: A probabilidade de um evento é igual à sua frequência de ocorrência em um grande número de experimentos:
  - \(P(A) = \lim_{n\to \infty} n_A/n\)
  - \(n_{A}\): número de vezes que o evento A ocorre em n experimentos.
  - Não é uma definição tão rigorosa porque é necessário provar que o limite existe
  - Essa interpretação permite realizar simulações
- Probabilidade da união de dois eventos: \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\)
  - \(\equiv P(A \cup B^{c}) \cup P(A \cap B) \cup P( A^{c} \cup B )\)
  - Como não possuem intersecção, torna-se a soma das probabilidades em que \(P(A) = P(A \cup B^{c}) + P(A \cap B)\)
  - O termo negativo evita a dupla contagem
- Teoremas:
  - \(P(A) = 1 - P(A^{c})\)
  - Se \(A \subset B\), então \(P(A) \leq P(B)\)
    - \(B = A \cup (A^{c} \cap B)\)

* Probabilidade condicional e Teorema de Bayes

** Probabilidade condicional

Sejam dois eventos \(A\) e \(B\) de uma mesmo espaço amostral \(\Omega\).
A probabilidade condicional de \(A\) dado que \(B\) ocorreu é dada por:

\[P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{P(A, B)}{P(B)} \quad P(B) > 0\]

Intuitivamente, é definido como a probabilidade de \(A\) ocorrer dado que o espaço amostral ficou delimitado à B.
A probabilidade de \(A\) ocorrer é dada por \(A \cap B\).

Exemplo: Se \(A\) é a probabilidade de sair \(2\) e \(B\) são os números pares de \(1 \ldots 10\):

\[P(A|B) = \frac{P(A \cap B)}{P(B)} = \frac{\frac{1}{10}}{\frac{5}{10}} = 0.2\]

O exercício com a tabela de contagem é bastante útil para aplicar diretamente a probabilidade condicional.

** Eventos independentes

Os eventos \(A\) e \(B\) são eventos independentes se, e somente se:

\[P(A \cap B) = P(A) P(B) \Leftrightarrow P(A | B) = P(A) \Leftrightarrow P(B | A) = P(B)\]
\[P(A|B) = P(A \cap B)/ P(B) = P(A) \Rightarrow P(A \cap B) = P(A) P(B)\]

Em outros termos, a ocorrência de \(B\) não altera a probabilidade de \(A\).
Tais propriedades permite testar se dois eventos são independentes.

** Teorema de Bayes


De acordo com o próprio Bayes, inferência de dados nada mais é do que a revisão de uma probabilidade (opinião metódica) com novas informações relevantes.

*** Partição do espaço amostral

- Partição: Considere alguns eventos \(B_{1}, \ldots, B_{n}\) que formam a partição do espalo amostral \(\Omega\):
  - Intersecção entre eles é vazio (\(B_{i} \cap B_{j} = \empty\))
  - Unição desses eventos é o espaço amostral (\(\cup_{i}^{n} B_{i} = \Omega\))
  - Todos tem \(P(B_{i}) \geq 0\)
  - Exemplo: Pensar em um quebra-cabeça
- Considere agora outro evento (\(A\)) composto por partes dessas partições
  - \(A = (A \cap B_{1}) \cup (A \cap B_{2}) \cup \ldots (A \cap B_{n})\)
  - O teorema é que se \(A\) for um evento do espaço amostral \(\Omega\), ele pode ser escrito tomando as partições desse espaço amostral
    - \(A = A \cup_{i}^{n} B_{i} \Rightarrow P(A) = P(A \cup_{i}^{n} B_{i})\)
    - Como esses eventos são mutuamente exclusivos
    - \(P(A) = \sum P(A \cap B_{i})\)
    - \(P(A \cap B_{i}) = P(A | B_{i}) \cdot P(B_{i}))\)
    - \(P(A) = \sum_{i=1}^{n} P(A | B_{i}) \cdot P(B_{i}))\)
- Teorema da probabilidade total: \(P(A) = \sum_{i=1}^{n} P(A | B_{i}) \cdot P(B_{i}))\)

** Demonstração do Teorema de Bayes

A partir da probabilidade total, pode-se anunciar o Teorema de Bayes.
Seja \(B_{1}, \ldots, B_{n}\) as partições de um espaço amostral; seja \(A\) um evento deste espaço amostral, então:

\[P(B_{i} | A) = \frac{P(A | B_{i}) P(B_{i}){{\sum_{j=1}^{n} P(A | B_{j}) \cdot P(B_{j})} \equiv \frac{P(A \cap B_{i})}{P(A)}\]

Com isso, a informação de que \(A\) ocorreu atualiza a probabilidade de que o evento \(B_{i}\) possa ocorrer.

* Variáveis aleatórias
** Introdução

Considere o lançamento de uma moeda por vezes sucessivas.
Considere ainda que exista uma observação específica de interesse (ex: número de caras).
Associe, em seguida, as probabilidades (\(p\) e \(1 - p\)) associadas a cada uma das possibilidades.
Neste caso, está definindo o valor da variável aleatória para cada saída do experimento.
Sendo assim, associa-se a cada saída à reta real a cada elemento do espaço amostral.

Convenção: Utiliza-se letras maísculas para denotar as variáveis aleatórias, enquanto as letras minúsculas são os valores dessas V.A.s.
Sendo assim, existe uma diferença entre o que será medido (variável aleatória (\(P(X \geq y)\))) e do valor observado (não aleatório, \(P(x \geq y)\), incorreto).

Resumidamente, uma variável aleatória é uma função que mapeia cada elemento do espaço amostral na reta real.

** Tipos de variáveis aleatórias

- Variáveis aleatórias discretas: Valores possíveis assumem valores discretos (podem ser listados, mesmo que a lista seja infinita).
  - Função discreta de probabilidade: associa a cada valor da VA a respectiva probabilidade (distribuição de probabilidade)
    - \(P(X = x_{i}) = p(x_{i})\)
    - \(\sum^{n}_{i} p(x_{i}) \equiv 1\)
    - Também chamada de Função Massa de Probabilidade
- Variável aleatória contínua: Se existir uma função de densidade de probabilidade (fdp) de \(X\) que satisfaça:
  - \(f(x) \geq 0 \forall x \in \mathbb{R}\)
  - \(\int_{-\infty}^{\infty} f(x) dx = 1\)
  - \(P(a \leq X \leq b) = \int_{a}^{b} f(x) dx \quad -\infty < a < b < \infty\)
  - \(f(x)\) é uma função com valores estritamente positivos e com área unitária

** Probabilidade condicional

Seja \(X\) uma variável aleatória (discreta ou contínua).
A probabilidade condicional de que \(X \in S\) dado que \(X \in V\) é dada por:

\[P(X \in S | X \in V) = \frac{P(X \in S \cap X \in V)}{P(X \in V)}\]
em que \(S\) e \(V\) são regiões da reta real.

*** Exemplo

Seja \(X\) uma variável aleatória cuja f.d.p. é:

\[f(x) = \begin{cases} 2x \quad 0 < x < 1 \\ 0 \quad \text{ c.c. } \end{cases}\]

Calcule \(P(X > \frac12 | X > \frac14)\):

- A interseccção entre as duas regiões é dado por \(P(X > \frac12)\)
- \(P(X > \frac12) \Rightarrow X^{2}\) avaliado entre \(1\) e \(\frac12\): \(\frac34\)
- \(P(X > \frac14) \Rightarrow X^{2}\) avaliado entre \(1\) e \(\frac14\): \(15/16\)
- \(P(X > \frac12 | X > \frac14) = P(X > \frac12 \cap X > \frac14)/P(X > \frac14) = \frac34/15/16 = 4/5\)

* Esperança e variância

** Valor esperado

- Variável aleatória discreta: \(E[X] = \sum^{\infty}_{i=1} x_{i}P(X = x_{i})\)
  - Exemplo: O valor esperado da face de um dado é \(3.5\) que, por sua vez, não é um valor discreto
    - De forma equivalente, é o valor médio observado considerando um número grande de simulações
  - Outro tipo de aplicação seria o cálculo do lucro médio sob incerteza
  - Função de variável aleatória discreta (\(g(x)\)): \(E[g(X)] = \sum^{\infty}_{i=1} g(x_i)P(X = x_{i})\)
    - Exemplo: \(g(X) = X^{2} \Rightarrow E[g(X)] = \sum x^2 P(x_i)\)
- Variável aleatória contínua: \(E[X] = \int_{-\infty}^{\infty}x f(x)dx\)
  - Função de variável aleatória continua (\(g(x)\)): \(E[g(X)] = \int_{-\infty}^{\infty}g(x) f(x)dx\)
- Notar que a esperança retorna o valor esperado e não é, por consequência, restrito às mesmas condições de uma probabilidade

*** Propriedades

- Se \(X = c\), então o valor esperado é constante (\(E[X] = c\))
- Se \(g(X) = cX\), então o valor esperado é constante (\(E[cX] = cE[X]\))
- Se \(g(X) = aX + b\), então o valor esperado é constante (\(E[aX] = aE[X] + b\))
- Exemplo: Seja \(E[X] = 2\). Calcule \(E[2X + 1] = \).
  - Resposta: \(2E[X] + 1 = 5\)

** Momento estatístico

Seja uma \(X\) uma variável aleatória discreta.
O momento estatístico de ordem \(n\) é dado por:

\[E[X^{n}] = \sum^{\infty}_{i=1} X^{n}P(X = x_{i})\]

Já para o caso de uma variável aleatória contínua:
\[E[X^{n}] = \int_{-\infty}^{\infty} x^n f(x)dx\]

** Momento central

O momento central de ordem \(n\) é dado por:

\[\mu_{n} = E[(X - E[X])^{n}] = \sum (x_{i} - E[X])^{n} P(X = x_{i})\]

\[\mu_{n} = E[(X - E[X])^{n}] = \int_{-\infty}^{\infty} (x - E[X])^{n}f(x)dx\]



Chama-se momento central por centralizar a variável aleatória em torno da média.

** Variância

A variância é o segundo momento central, dado por:
\[V(X) = \sigma^{2} = E[(X - E[X])^{2}]\]
enquanto o desvio-padrão (\(\sigma\)) é a raíz da variância.
Sendo assim, a variância é a média das distâncias Euclidianas em relação à média.
De forma equivalente:
\[V(X) = E[X^{2}] - E^{2}[X]\]

\[V(g(X)) = E[g(X)^{2}] - E^{2}[g(X)]\]


Propriedades:

\[V(aX + b) = a^{2} V(X)\]

Exemplo: \(V(2X + 5) = 4V(X)\). Seja \(V(X) = 4 \Rightarrow V(2X + 5) = 16\).
* Função de Distribuição acumulada
** Definição

A função de distribuição acumulada (\(F(X)\))  de uma variável aleatória é definida por: \(F(x) = P(X \leq x)\).
- Para o caso discreto: \(F(x) = P(X \leq x) = \sum_{x_i \leq x} P(X = x_i)\) (somar até o valor).
- Para o caso contínuo: \(F(x) = P(X \leq x) = \int_{-\infty}^{x} f(s)ds\).
- Alguns intervalos devem ser definidos para computar (e plotar) a acdf.
  - Apesar a distribuição poder assumir valores discretos, a função de distribuição é um valor contínuo
- Vale notar que também pode ser referida como função de distribuição.
  - Notar também que existe diferença entre distribuição de probabilidade (maepeia espaço amostral à probabilidades) da função de distribuição (que mapeia a probabilidade de forma acumulada).
    - Isso permite calcular derivadas, por exemplo, dado que \(X\) assume valores contínuos
      - A função, por sua vez, é descontínua dado que possui intervalos abertos

Propriedades:
- \(0 \leq F(X) \leq 1\)
- \(F(X)\) é não decrescente
- \(\lim_{x \to -\infty} F(X) = 0\)
- \(\lim_{x \to \infty} F(X) = 1\)
- Calculando a função densidade de probabilidade (pdf)
  - Caso discreto: \(P( a \leq X \leq b ) = F(b) - F(a)\)
  - Caso contínuo: \(f(x) = \frac{d}{dx} F(x)\)
    - Usando o teorema do valor médio da integração: \(\int_{x}^{x + \Delta x} f(s) ds = f(c) + \Delta x\)
* Modelos probabilísticos discretos
** Introdução


Diferentemente de um modelo determinístico, um modelo aleatório indica a probabilidade de estar em determinado estado.
Uma vez que o experimento aleatório é repetido um número grande de vezes, emerge-se um padrão que, por sua vez, permite calcular a probabilidade associada a cada evento.
Ao longo desta aula, está sendo usado o termo modelo e distribuição de forma equivalente.

** Distribuição uniforme discreta

Uma variável aleatória \(X\) segue uma distribuição/modelo uniforme se atribui a mesma probabilidade de todos os eventos discretos ocorrerem.
A distribuição de probabilidade é dada por:
\(P(X = x) = 1/n\) em que \(n = 1 + (b - a)/c\)

- Esperança: \(E[X] = (a + b)/2\)
- Variância: \(V(X) = c^{2}(n^{2}-1)/12\)
  - Em que \(c\) é o tamanho (constante) do incremento (\(c = (b-a)/(n-1)\))

** Distribuição de Bernoulli

Diz-se que uma variável aleatória segue o modelo de Bernoulli se atribui \(0\) ao fracasso de um evento e \(1\) ao sucesso.
Adotando a convenção de que a probabilidade de sucesso é denotado por \(p\), a distribuição de probabilidade é dada por:
\(P(X = k) = p^k(1 - p)^{1-k} \quad k = 0,1\)
Neste caso, ocorre apenas uma ocorrência.

- Esperança: \(E[X] = p\)
- Variância: \(V(X) = p(1-p)\)

** Distribuição de Binomial

Esta é a generalização da distribuição de Bernoulli com \(n\) repetições, ou seja, é um processo estocástico de Bernoulli.[fn:processo-estocastico]
Propriedades:
- Cada tentativa pode ser atribuída como sucesso ou falha da mesma forma
- Probabilidade de sucessão se mantém constante
  - Isso equivale ao experimento ser com reposição (garante que \(p\) seja constante)
- Tentativas são independentes

A probabilidade de obter \(k\) sucessos em \(n\) repetições é dado por: \(P(X = k) = C^{k}_{n} p^k(1 - p)^{n-k} \quad k = 0,1\).
Em que \(C^{k}_{n}\) é a combinação de \(n\) elementos tomados de \(k\) em \(k\):
\(C^k_n = \frac{n!}{(n-k)k!}\).
Neste caso, a ordem dos experimentos não é relevante.
Caso o número de sucessos seja condicional (em termos de "pelo menos", "entre", etc), utiliza-se o somátória em termos de \(k\).

- Esperança: \(E[X] = np\)
- Variância: \(V(X) = np(1-p)\)
- Interpretação: \(P(X = k)\) representa a probabilidade de obter \(k\) sucessos em \(n\) tentativas

[fn:processo-estocastico]: Processo probabilístico repetido ao longo do tempo.

** Distribuição de Poisson

Uma variável aleatória discreta \(X\) segue o modelo de Poisson com taxa \(\lambda >0\) se:
\(P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!} \quad k = 0, 1, \ldots,\)

Essa é uma distribuição quando possui informação sobre a taxa de ocorrência (capturado por \(\lambda\)) enquanto \(k\) caputura o número de ocorrências.
Tipicamente, representa fenômenos em que a probabilidade aumenta com o número de repetições (quanto mais tempo espera-se pelo ônibus, maior a chance do ônibus chegar).

- Esperança: \(E[X] = \lambda\)
- Variância: \(V(X) = \lambda\)

*** Lei dos eventos raros

Estabelece a relação entre a distribuição binomial e a distribuição de Poisson.
Quando o número de repetições converge ao infinito e a taxa de ocorrência é constante (\(\lambda = np\)), a distribuição Binomial converge para a distribuição de Poisson.
Isso se deve porquê \(\lambda = np\), com \(\lambda\) sendo constante e \(n \to \infty\), tem-se que \(p \to 0\).
Por essa razão, trata-se de uma relação com eventos raros.

** Distribuição geométrica

Trata-se de uma distribuição em que observa-se a primeira ocorrência de sucesso.

\(P(X = k) = p(1 - p)^{k-1} \quad k = 1,2,3,\ldots\)

Neste caso, \(X\) representar o número de falhas até o primeiro sucesso.
Sendo assim, é o número de experimentos, não de sucessos.
Em R, é preciso incluir o zero até \(k-1\) para encontrar pelo menos \(k\) repetições.


- Esperança: \(E[X] = 1/p\)
- Variância: \(V(X) = (1-p)/p^{2}\)
- Interpretação: \(P(X = k)\) representa a probabilidade do primeiro sucesso ocorrer no experimento \(k\)

** Distribuição binomial negativa

Seja \(X\) o número de repetições necessárias a fim de que ocorram exatamente \(r\) sucessos de modo que o \(r\)-ésimo sucesso ocorra na \(k\)-ésima tentativa.

\[P(X = k) = {k-1}\choose{r-1} p^{r-1}(1 - p)^{k-r}p, \quad k = r, r+1, \ldots\]

Neste caso, a última repetição (\(k\)) é necessariamente um sucesso.

De forma mais intuitiva, esse modelo indica o caso em que a última observação é necessariamente um sucesso, enquanto as anteriores podem ser sucesso ou falha.


- Interpretação: \(P(X = k)\) representa a probabilidade de serem necessários \(k\) experimentos para obter \(n\) sucessos; \(n-1\) sucessos ocorreram em \(k-1\) tentativas anteriores e o \(n-\)ésimo sucesso ocorre na tentativa \(k\).
  - Exemplo: Time de futebol vencer determinada série em \(n\) partidas (em que é um número menor do que o número total de partidas)

** Distribuição hipergeométrica

Considere um conjunto de \(N\) objetos, dos quais \(N_{1}\) são do tipo 1 e os demais são do tipo 2.
Para um sorteio de \(n\) objetos (\(n < N\)) *sem* reposição, seja \(X\) a variável aleatória que define o número de
objetos do tipo 1 sorteados.
Então, a probabilidade de sortearmos \(k\) objetos do tipo 1:


\[P(X = k) = \frac{{N_1}\choose{k}}{{N} {N - N_1}\choose{n - k}}{{N}\choose{n}}, \quad k = 0, 1, \ldots, n\]

- Interpretação: \(P(X = k)\) representa a probabilidade de obter \(k\) sucessos em \(n\) tentativas sem repetição

** Exercícios

*** Questão 01

Uma urna tem 20 bolas pretas e 30 brancas. Retiram-se 25 bolas com reposição. Qual a probabilidade de que:

a) 2 sejam pretas?
b) pelo menos 3 sejam pretas?

* Modelos probabilísticos contínuos

Lembrando que esses modelos informam qual é a distribuição de determinada variável aleatória.
Isso permite combinar exercícios com diferentes distribuições.
Exemplo: Uma variável aleatória segue uma distribuição normal; qual é a probabilidade de que determinado item em uma linha de produção seja o primeiro rejeitado em uma sequência de \(n\) ocorrências (distribuição geométrica).
Seria o caso de que \(P(X < x)\) e \(P(Y = k)\) (combinando modelos fdp discretas com modelos discretos).

** Distribuição uniforme

Uma variável aleatória contínua segue uma distribuição uniforme se sua função densidade de probabilidade possui uma probabilidade constante em um intervalo fechado \(\[a,b\]\).
Intervalos com um mesmo comprimento terão a mesma probabilidade.

- Esperança: \(E[X] = (a + b)/2\)
- Variância: \(V(X) = (b-a)^2/12\)
  - Em que \(c\) é o tamanho (constante) do incremento (\(c = (b-a)/(n-1)\))

** Distribuição normal

Uma variável aleatória contínua que tome todos os valores na reta real segue uma distribuição Gaussiana se sua função densidade de probabilidade possui a seguinte forma:

\[f(x) = \frac{1}{\sigma\sqrt{2\pi}}\exp \left[-\frac{1}{2}\left(frac{x - \mu}{\sigma}\right)^{2}\right], \quad -\infty < x < \infty\]
- Esperança: \(E[X] = \mu\)
- Variância: \(V(X) = \sigma^{2}\)
- Propriedades:
  - Simétrica em relação à \(\mu\)
  - \(f(x) \to 0\) quando \(x \to \pm \infty\)
  - O valor máximo de \(f(x)\) ocorre quando \(x = \mu\)

Se \(X \sym \mathbb{N}(\mu, \sigma^{2})\), então se \(Y = aX + b\), então:
\[Y \sym \mathbb{N}(a\mu + b, a^{2}\sigma^{2})\]

O mesmo raciocínio se aplica para definir a normal padronizada: \(Z \sym \mathbb{N}(0,1)\) em que \(Z = \frac{X -\mu}{\sigma}\).
Para computar \(P(a \leq X \leq b) \int_{a}^{b} f(x)dx\) quando \(X\) segue uma distribuição normal, pode não ter valor exato, então adota-se o método de resolução numérica.
Sendo assim, transforma-se \(X\) em \(Z\) a compara com o valor já tabelado de modo que \(P\left(\frac{a - \mu}{\sigma} \leq Z \leq \frac{b - \mu}{\sigma} \right)\).

Exemplo: Se \(X \sym \mathbb{N}(\mu = 165, \sigma^{2} = 9)\), calcule \(P(X < 162)\):

\[P(X < 162) = P\left(Z < \frac{162 - 165}{\sqrt{9}}\right) = P(Z < -1)\]

Olhando os dados tabelados, \(P(Z < -1) = 0.158\) dado que essas tabelas indicam a probabilidade \(< a\), sendo a área abaixo da curva normal até o ponto \(a\), sem incluí-lo.
Em programação, utiliza-se funções que retornam CDF (distribuição acumulada, \(F(a)\)).

Para os casos \(P(X > b)\), faz-se \(1 - P(X < b)\).
Com \(P(a \leq X \leq b) = P(X \leq b) - P(X \leq a)\)

** Distribuição exponencial

Uma variável aleatória \(X\) segue o modelo exponencial se sua função de densidade de probabilidade é dada por:


\[f(x) = \lambda \exp{-\lambda x}, x \geq 0; 0 c.c. \quad \lambda > 0\]


- Quando \(x = 0, f(x) = \lambda\)
  - Quando maior \(\lambda\), maior o decaimento
  - Interpretação é a de taxa de "emissão"
  - O tempo de espera segue um modelo exponencial
    - Como consequência, implica que o sistema não tem memória
    - Não é possível prever o futuro a partir dos dados do passado
    - Também são chamados de sistemas Markovianos
    - \(P(X > t + s | X > s) = P(X > t)\)
- Esperança: \(E[X] = 1/\lambda\)
  - Sabendo qual é o tempo de vida médio \(E[X]\), pode-se obter o \(\lambda\) como consequência
- Variância: \(V(X) = 1/\lambda^{2}}\)

** Distribuição gamma


Uma variável aleatória \(X\) segue o modelo gamma se sua função de densidade de probabilidade é dada por:

\[f(x) = \frac{\lambda^{\alpha}}{\Gamma(\alpha)}x^{\alpha-1} e^{-\lambda x} \quad \lambda, \alpha > 0, x \geq 0; 0 c.c.\]
em que
\[\Gamma(\alpha) = \int_{0}^{\infty} t^{\alpha-1} \exp^{-t}dt\]

Se \(\alpha\) for inteiro positivo, equivale à distribuição de Erlang.
A distribuição de Erlang corresponde a soma de \(\alpha\) variáveis aleatórias independentemente distribuídas que, por sua vez, seguem a distribuição exponencial.

- Esperança: \(E[X] = \alpha/\lambda\)
- Variância: \(V(X) = \alpha/\lambda^{2}}\)

** Outras distribuições

- Distribuição chi-quadrado :: Resultante de soma de \(k\) variáveis aleatórias independentes com distribuição zero e desvio-padrão unitário, possuindo \(k\) grau de liberdade.
  - \[X = \sum^k Z_{i}^{2}\]
  - \[X \sym \chi^{2}_{k} \]
- Distribuição beta :: Trata-se dos casos em que a variável aleatória contínua é limitada entre 0 e 1
  - Possui uma função Beta que atua como constante de normalização
  - Utilização, por exemplo, quando a V.A. é representada por porcentagens de ocorrências
- T de student :: Esta distribuição se aproxima de uma distribuição normal quando se aumenta os graus de liberdade

* Variáveis aleatórias multidimensionais

** Introdução

Seja \(\epsilon\) um experimento aleatório associado a um espaço amostral \(\Omega\).
Sejam \(X_{1} = X_{1}(\omega), X_{2} = X_{2}(\omega), \ldots\), funções que associam um número real  a cada resultado \(\omega \in \Omega\).
Denominamos vetor aleatório o conjunto \(X = [X_{1}, X_{2}, \ldots]\).
Sendo assim, invés de associar cada ponto no espaço amostral à reta real, associa-se a um espaço \(n\)-dimensional.


Sejam \(X\) e \(Y\) variáveis aleatórias associadas a um espaço amostral \(\Omega\).
O par \((X,Y)\) será uma variável aleatória discreta bidimensional se os valores possíveis forem finitos ou infinito enumeráveis.
A cada resultado possíve associamos um número

\[p(x_{i},y_{j}) = P(X = x_{i}, Y = y_{j})\]
Esta é a distribuição de probabilidade conjunta discreta, satisfazendo:
- \(0 \leq P(X = x_{i}, Y = y_{j}) \leq 1 \forall i,j, = 1, 2, \ldots\)
- \(\sum_{i}^{\infty} \sum_{j}^{\infty} P(X = x_i, Y = y_j) = 1\)

Estendendo para o caso contínuo, tem-se que \((X,Y)\) devem tomar todos os valores possíveis em algum conjunto não enumerável de modo que a função densidadede probabilidade conjunta satisfaz:

- \(f(x,y) \geq 0 \forall (x,y) \in \mathbb{R}^{2}\)
- \(\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x,y)dxdy = 1\)

** Distribuição de probabilidade marginal

Seja \((X, Y)\) uma variável aleatória discreta bidimensional.
Sendo assim, a distribuição de probabilidade marginal de \(X\) é dada por:

\[P(X = x_{i}) = \sum_{j=1}^{\infity} P(X = x_i, Y = y_j)\]

enquanto a de \(Y\) é:
\[P(Y = y_{j}) = \sum_{i=1}^{\infity} P(X = x_i, Y = y_j)\]

Como exemplo, tome a distribuição de probabilidade conjunta discreta:

| \(X_{1}, X_{2}\) | 0    | 1    |
|          0 | 9/21 | 5/21 |
|          1 | 5/21 | 2/21 |


A distribuição de probabilidade marginal é dada por:


|    \(X_{1}, X_{2}\) | 0     | 1    | \(P(X_{1} = x)\) |
|             0 | 9/21  | 5/21 | 14/21         |
|             1 | 5/21  | 2/21 | 7/21          |
| \(P(X_{2} = x)\) | 14/21 | 7/21 | 1             |

Interpretação \(P(X_{2} = 1)\): \(P(X_{2} = 1, X_{1} = 0) + P(X_{2} = 1, X_{1} = 1) = 7/21\)
Sendo assim, é a probabilidade quando fixa-se uma das variáveis enquanto as demais podem variar livremente.

No caso contínuo, temos:

\[f_x(x) = \int_{-\infty}^{\infty} f(x,y)dy\]
\[f_y(y) = \int_{-\infty}^{\infty} f(x,y)dx\]

Isso permite, por exemplo, calcular \(P(a \leq X \leq b)\) a partir da densidade marginal de \(X\):

\[P(a \leq X \leq b) = \int_{a}^{b} f_{x}(x)dx\]
Sendo assim, é possível avaliar a probabilidade de apenas uma das variáveis a partir da função densidade de probabilidade marginal.

** Probabilidade condicional
:LOGBOOK:
CLOCK: [2026-04-20 Mon 10:37]--[2026-04-20 Mon 11:29] =>  0:52
:END:

Seja o vetor aleatório bidimensional \((X,Y)\).
A probabilidade condicional de \(X = x\) dado que \(Y = y\) é dada por:

\[P(X = x, Y = y) = \frac{P(X = x, Y = y)}{P(Y = y)} \quad P(Y = y) > 0\]

Sendo assim, é a função de densidade de probabilidade conjunta dividida pela função de densidade marginal.

|    \(X_{1}, X_{2}\) | 0     | 1    | \(P(X_{1} = x)\) |
|             0 | 9/21  | 5/21 | 14/21         |
|             1 | 5/21  | 2/21 | 7/21          |
| \(P(X_{2} = x)\) | 14/21 | 7/21 | 1             |

Calculando que \(P(X_{1} = x | X_{2} = y)\), tem-se:

|                                     | X2 = 0     | X2 = 1    |
| \(P(X1 = 0  \text{ dado } X2 = x)\) | 9/21/14/21 | 5/21/7/21 |
| \(P(X1 = 1  \text{ dado } X2 = x)\) | 5/21/14/21 | 2/21/7/21 |


Para o caso contínuo, temos:

\[f_{X|Y}(x | Y = y) = \frac{f(x, y)}{f_Y(y)} \quad f_Y(y) > 0\]

Para uma situação em que \(X\) varia entre \([0,1]\), tem-se:
\[P(X \geq \frac12 | Y = y) = \int_{\frac12}^{1}f_{X|Y}(x|y)dx = \int_{\frac12}^{1} \frac{f(x,y)}{f_Y(y)} dx \]

** Independência

As variáveis aleatórias são independentes se, para cada caso:
- Discreto :: \(P(X = x_i, Y = y_j) = P(X = x_i)P(Y = y_j) \forall i\neq j\)
- Contínuio :: \(f(x,y) = f_X(x)f_Y(y) \forall (x,y) \in \mathbb{R}^{2}\)

Caso elas sejam idependentes, não é possível utilizar nenhuma delas para predizer a outra.

* Esperança e variância condicionais (caso multivariado)

** Esperança de um vetor aleatório

Seja \(g(X,Y)\) uma função das variáveis aleatórias \(X\) e \(Y\)

- Caso discreto :: \(E(g(X,Y)) = \sum^{\infty}_{i=1}\sum^{\infty}_{j=1}g(x_i, y_j)P(X = x_i, Y = y_j)\)
- Caso contíuo :: \(E(g(X,Y)) = \int^{\infty}_{-\infty}\int^{\infty}_{-\infty}g(x_i, y_j)P(X = x_i, Y = y_j)dxdy\)
  - Em que \(P(\dot)\) é a função densidade de probabilidade
- Propriedades:
  - Se \(X\) e \(Y\) forem independentes, então \(E[XY] = E[X]E[Y]\)
    - No entanto, a volta não é válida (a identidade da multiplicação não defini independência)

*** Esperança condicional


- Caso discreto: \(E[X | Y = y] = \sum^{\infty}_{i=1} x_{i}P(X = x_{i} | Y = y)\)
- Caso contínuo: \(E[X | Y = y] = \int_{-\infty}^{\infty}x f_{X|Y}(x|y)dx\)
- Em que \(P(X = x_{i} | Y = y)\) e \(f_{X|Y}(x|y)\) são probabilidades condicionais
  - Memo: \(f_{X|Y}(x|y) = f(X,Y)/f_{Y}(Y)\)

*** Lei da esperança total

\[E(X) = E(E(X | Y))\]

- Caso discreto: \(E(X) = \sum_{y} E(X | Y = y) P(Y = y)\)
  - Exemplo discutido: Trabalhador tem vários caminhos, mas alguns deles retorna ao ponto inicial
    - Sendo assim, a esperança aparece também na probabilidade desses caminhos que envolvem o retorno
  - Sendo assim, mostra-se como pode estender o cômputo da esperança total para o caso multivariado
    - Raciocínio similar foi utilizado ao aplicar o Teorema de Bayes

** Variância de um vetor aleatório

Tal como no caso univariado,

\[V[g(X,Y)] = E[g(X, Y)^{2}] - E^{2}[g(X, Y)]\]

- Propriedades:
  - Se \(X\) e \(Y\) forem independentes, então \(V(X + Y) = V(X) + V(Y)\)

*** Variância condicional

\[V[X|Y] = E[X^{2} | Y] - E^{2}[X | Y]\]

** Covariância

Sejam \(X\) e \(Y\) variáveis aleatórias, a covariância entre elas é dada por:

\[Cov(X, Y) = E((X - E(X))(Y - E(Y))) = E(XY) - E(X)E(Y)\]

A covariância mostra a relação entre duas variáveis.

Caso sejam independentes, a covariância é nula.
Note que:
\[Cov(X, X) = E((X - E(X))(X - E(X))) = E(X^{2}) - E^{2}(X) = V(X)\]

** Correlação de Pearson

Essencialmente, é a covariância normalizada pela multiplicação dos desvios-padrão:

\[\rho_{X,Y} = \frac{cov(X,Y)}{\sqrt{V(X)V(Y)}} = \frac{cov(X,Y)}{\sigma_X \sigma_Y}\]

Como consequência, a covariância é remapeada entre \[[-1,1]\].

Caso seja calculado para uma *amostra* de dados, tem-se o coeficiente de Pearson (\(r\)), com as mesmas delimitações da correlação populacional.

É importante destacar que a correlação de Pearson só é capaz de capturar relações *lineares*.
Além disso, esse coeficiente não mede a inclinação.
Caso a inclinação seja nula, o coeficiente de Pearson será zero; caso contrário, pode assumir qualquer outro valor.
Por fim, vale lembrar que não mede causalidade (apenas correlação).

** Distribuição normal multivariada

Uma vez definida a correlação de Pearson, é possível apresentar a distribuição normal multivariada.

f(\mathbf{x}) = \frac{1}{\sqrt{(2\pi)^k |\boldsymbol{\Sigma}|}} \exp\left( -\frac{1}{2} \underbrace{(\mathbf{x} - \boldsymbol{\mu})' \boldsymbol{\Sigma}^{-1} (\mathbf{x} - \boldsymbol{\mu})}_{\text{Distância de Mahalanobis}^2} \right)
\end{equation}
#+end_src

em que::
- \(\mu\) é o vetor de médias
- \(\boldsymbol{\Sigma}\) é a matriz de covariância
  - Ao longo da diagonal principal, tem-se a variância de cada variável
  - Fora da diagonal principal, tem-se as covariâncias

OBS: A função densidade de probabilidade conjunta pode ser também avaliada como curvas de nível.
A inclinação das curvas de nível será dada pela correlação enquanto as variâncias individuais regulam o espaçamentos entre os níveis.
A relação entre as variâncias, por sua vez, dará informação sobre qual direção as curvas de nível serão mais alongadas.

** Introdução


A ideia da simulação de Monte-Carlo é reproduzir um experimento aleatório no computador.
Dito de outro modo, é um método para gerar números aleatórios com base em determinada função de probabilidade.

*** Gereção de números aleatórios

No método chamada Linear Congruential Generator, nós geramos uma sequência de números pseudo-aleatórios através da relação de recorrência:
$$X_{n+1} = (aX_n + c) \mod m$$
onde $X_0$ é a semente, \(m\), \(a\) e \(c\) são inteiros maiores do que zero.
A sequência obtida satisfaz  $0 \leq X_i \leq m$.
Para gerar o número no intervalo [0,1], usamos $$U_n = X_n/m.$$

#+begin_src python
import numpy as np
import matplotlib.pyplot as plt

# função para gerar os números aleatórios
def random(Xi, a, c, m):
    Xii = (a*Xi + c)%m
    return Xii

# constantes
a = 11; c = 0; m = 16
X0 = 1 # semente do gerador
n = 20 # quantidade de números a serem gerados
Xi = X0 # inicia a sequência de números
X = [X0] # armazena os números gerados
for i in range(n):

Notamos que a sequência é periódica, sendo que $[1, 11, 9, 3]$ se repete.
Ou seja, não temos um bom gerador de números aleatórios, pois podemos prever o próximo número se descobrir a sequência.
Para resolver esse problema, temos o seguinte teorema, demonstrado no livro The Art Of Computer Programming (Donald Knuth).

O gerador pseudo-aleatório linear produz uma sequência aperiódica se as seguintes condições forem satisfeitas:
- \(X_{n+1}_{} = (aX_n + c)\mod m, \quad U_n = X_n/m.\)
- Se $q$ é um número primo que divide $m$, então ele divide $b=a-1$.
  - OBS: \(q\) não está definido ainda
- Se $m$ é múltiplo de 4, então $b=a-1$ deve ser múltiplo de 4.
- O único inteiro que divide exatamente $m$ e $c$ é  valor um.

Simulando com valores que satisfazem essas condições:

*** Semente de geração de números aleatórios

Sempre que a definição da semente for declarada, o gerador de números aleatórios é reinicializado.
Isso permite que o experimento seja reproduzível.

** Cálculo de integrais

Podemos calcular integrais numericamente usando números aleatórios.
Temos:
$\theta = \int_{0}^{1} g(u)du = \int_{0}^{1} 1\cdot g(u)du = \int_{0}^{1} f(u) g(u)du = E[g(U)]$
onde $U$ tem distribuição uniforme em $[0,1]$:
\begin{equation}
f(u) =
\begin{cases}
1, \quad 0 \leq u \leq 1\\
0 \quad \mbox{caso contrário}.
\end{cases}
\end{equation}

Assim, podemos calcular $\theta$ numericamente:
$\lim_{n \rightarrow \infty} \sum_{i=1}^{n}\frac{g(U_i)}{n} = E[g(U)] = \theta$

Sendo assim, a integral é numericamente aproximada de uma soma que considera números aleatórios.
Vamos considerar um exemplo: $\theta = \int_{0}^{1}(x+1)dx = 1.5$

#+begin_src python
import numpy as np
# função a ser integrada
def g(x):
    f = x + 1
    return f
n = 1000 # número de valores sorteados
theta = 0
for i in range(n):
    # sorteia um valor uniformemente em [0,1]
    u = np.random.uniform(0,1)
    theta = theta + g(u)
# calcula a média de theta
theta = theta/n
print("Integral: ", theta)
#+end_src

No caso geral, temos as seguinte integral:
$\theta = \int_{a}^{b} g(u) du$
se fizermos $y = \frac{u-a}{b-a}$, temos $du = (b-a)dy$.
Basicamente, faz-se a troca de variáveis de modo à fazer a normalização min-max dos limites da integral, retornando ao caso anterior.
Assim,
$\theta = \int_{a}^{b} g(u) du = \int_{0}^{1} g[(b-a)y + a](b-a) dy = \int_{0}^{1} h(y) dy$,
onde $h(y) = (b-a)g[(b-a)y + a]$.
$\theta = E[h(Y)]$.
Vamos considerar um exemplo:
$\theta = \int_{2}^{4}(x+1)dx = 8$

#+begin_src python
import numpy as np
import matplotlib.pyplot as plt

# função a ser integrada
def g(x):
    g = x + 1
    return g

def h(y,a,b):
    val = (b-a)*g(a+(b-a)*y)
    return val

valor_teorico = 8
a = 2; b = 4 # limites de integração
Nmax = 1000 # número máximo de simulacoes
vtheta = []
vn = []
for n in range(1,Nmax):
    theta = 0
    for i in range(n):
        u = np.random.uniform(0,1)
        # calcula theta
        theta = theta + h(u,a,b)
    # calcula o valor médio de theta
    theta = theta/n
    vtheta.append(theta)
    vn.append(n)
# mostra os resultados
plt.figure(figsize=(8,6))
plt.plot(vn, vtheta, linestyle='-', color="gray",
         linewidth=2,label = 'Valor simulado')
plt.axhline(y=valor_teorico, color='black',
            linestyle='--', label = 'Valor teórico')
plt.ylabel('E[g(U)]', fontsize=20)
plt.xlabel("Número de simulações", fontsize=20)
plt.legend(fontsize = 15)
plt.savefig("./figs/Int2_MC.svg")
plt.show(True)
#+end_src

No caso em que um dos limites de integração é infinito, procedemos da seguinte maneira:
$\theta = \int_{0}^{\infty} g(u) du$
definindo: $y = 1/(x+1)$ temos $dy = y^2 dx$.
Então
$\theta = \int_{0}^{\infty} g(u) du = \int_{0}^{1}g(\frac{1}{y} - 1)\frac{dy}{y^2} = \int_{0}^{1}h(y)dy$
onde
$h(y) = \frac{g(1/y-1)}{y^2}$
$\theta = E[h(Y)]$.

Vamos calcular:
$\theta = \int_{0}^{\infty} e^{-x}dx = 1$

#+begin_src python
import random

def g(x):
    f = np.exp(-x)

# realiza a simulação
for n in range(1,Nmax):

** Integrais múltiplas

Exemplo: Calcule a integral a seguir usando números aleatórios.
$$
\theta = \int_0^2 \int_1^2 \left(\frac{x^2y}{3} + y^2\right)dydx = 6
$$

#+begin_src python
import numpy as np
import matplotlib.pyplot as plt

# função a ser integrada
def g(x,y):
    g = (x**2)*y/3 + (y**2)
    return g

def h(z,w,a, b, c , d):
    h = (b-a)*(d-c)*g(z*(b-a)+a,w*(d-c)+ c)
    return h

valor_teorico = 6
Nmax = 1000 # número máximo de simulações
vtheta = []
vn = []
# intevalos de integração
a = 0; b = 2; c = 1; d = 2
for n in range(1,Nmax):
    theta = 0
    for i in range(n):
        u1 = np.random.uniform(0,1)
        u2 = np.random.uniform(0,1)
        theta = theta + h(u1,u2,a,b,c,d)
    # calcula o valor médio de theta
    theta = theta/n
    vtheta.append(theta)
    vn.append(n)
# mostra os resultados
plt.figure(figsize=(8,6))
plt.plot(vn, vtheta, linestyle='-', color="gray",
         linewidth=2,label = 'Valor simulado')
plt.axhline(y=valor_teorico, color='black',
            linestyle='--', label = 'Valor teórico')
plt.ylabel('E[g(U)]', fontsize=20)
plt.xlabel("Número de simulações", fontsize=20)
plt.legend(fontsize = 15)
plt.savefig("./figs/int4_MC.svg")
plt.show(True)
#+end_src
